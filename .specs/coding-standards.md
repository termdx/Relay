# Coding Standards

## General

-   Feature-first folder structure
-   Dependency injection
-   Strong typing
-   Domain-driven modules

## Rules

1.  No business logic in controllers.
2.  No AI calls from controllers.
3.  Prefer events over tight coupling.
4.  Keep workflows idempotent.
5.  Write integration tests for workflows.
6.  Every module owns its domain.
7.  Keep interfaces stable.
